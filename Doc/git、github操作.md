## git操作

### 一. 初次配置
1. 配置署名和邮箱。没有--global则表明只影响当前文件夹
```
git config --global user.name "用户名"
git config --global user.email "邮箱"
```


2. 设置颜色高亮
```
git config --global color.ui true
```
